Reduce entropy as you dig

A messy pile only gets messier unless you fight back a little at every step. These habits make rebuilds faster and way less frustrating.

Show tipsHide tips
Sort by colour earlyEven rough colour buckets shrink the search space fast. You do not need perfect sorting, just fewer places to look.
Stack identical piecesFound two of the same part? Click them together right away. Future-you will not have to hunt the same shape twice.
Use bowls, plates, and sheet pansOne container per part type (or per colour) keeps found pieces from rejoining the rubble. Cookie sheets are underrated.
Corral the dropsBricks bounce. Work over a table, keep a tray on the floor, and tap the drop counter when one escapes. Less crawling, more digging.
Bring in the whole familyOne person reads the card, one person searches the pile, one person stacks. Kids are surprisingly good spotters.
Perfect stuck-inside daysToo hot, too icy, or just a lazy Saturday? Rebuilding from the tub is a great indoor project with something to show for it.
3D print the odd missing pieceWeird hinge, cracked clip, or a rare colour? Printing a stand-in can finish the set while you keep hunting the real one.
Check the value (optional)Some childhood sets have climbed in price. Knowing what you are rebuilding can make the hunt feel even sweeter.

📋 BrickLink paste

Open a set's Item Inventory, select all (Ctrl/Cmd+A), copy, paste above. Include the Minifigures section too.

1 3004 Black Brick 1 x 2 1 sp043 UFO Droid - Blue…

Get it: BrickLink → set page → Item Inventory

📊 Brickset CSV

Export a set's parts list from Brickset. Minifigs are auto-detected when Category says Minifigures.

Part,Color,Quantity,Category 3001,Bright Red,6,Parts sw0001,Luke Skywalker,1,Minifigures

Get it: Brickset → set → Export → CSV

📝 Your own CSV / spreadsheet

Add a header row — columns can be in any order. Only quantity and a part name or number are required.

  • Qty — qty, quantity, count, need
  • Name — name, description, part name
  • Part # — part, part_num, itemid
  • Colour — color, colour (auto-swatched)
  • Hex — rgb, hex (optional)
  • Image — img, img_url (optional)

No header? Assumes part, name, color, quantity by position.

📄 BrickLink XML & plain text

Upload a .xml inventory export, or tab-separated rows. Lines like Brick 2 x 4 x6 work without headers.
